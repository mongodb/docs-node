===============
Quick Reference
===============

.. default-domain:: mongodb

This page demonstrates the syntax of a list of MongoDB commands
and links to related reference and API documentation.

.. list-table::

       - `API Documentation <{+api}/classes/Collection.html#findOne>`__
       - :ref:`Usage Example <node-usage-findone>`
       - :ref:`Fundamentals <node-fundamentals-retrieve-data>`

     - .. io-code-block::
          :copyable: true

          .. input::
             :language: js

             coll.findOne({ title: 'Hamlet' });

          .. output::
             :language: js

             { name: 'Hamlet', type: 'movie', ... }

       - `API Documentation <{+api}/classes/Collection.html#find>`__
